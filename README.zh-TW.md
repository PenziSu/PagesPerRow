# PagesPerRow

[English](README.md) | **繁體中文**

一款圍繞單一概念打造的 PDF 閱讀器：一次看很多頁，全部排在同一列。

大多數 PDF 閱讀器一次只顯示一兩頁。PagesPerRow 讓你自訂每一列要並排幾頁 —— 4 頁、6 頁、8 頁，螢幕塞得下多少都行 —— 於是你能一眼掌握整篇論文，而不是一頁一頁慢慢捲。這是一種鳥瞰式的閱讀模式，給那些習慣先在空間上理解一份文件、再線性細讀的人。

它就是**一個 HTML 檔**。免安裝、免建置、免伺服器、免註冊。打開它、把 PDF 拖進去，就這樣。你的檔案不會離開你的電腦 —— 一切都在瀏覽器本機執行。

## 為什麼會有這個專案

如果你習慣用 PDF 讀學術論文，大概曾想快速掃過整體結構 —— 圖表在哪、各章節多長、參考文獻從哪開始 —— 又不想被迫一頁一頁線性爬。現有工具其實都給不了這個：

- **Adobe Acrobat** 有縮圖側欄，但那只是一條窄窄的導覽帶，不是你能實際工作於其中、可閱讀的高密度版面。
- **sioyek**，這款在教科書與論文圈很受歡迎的閱讀器，只支援單頁與雙頁模式。多個 open issue 要求「每列三頁以上」（相當於 Zathura 的 `pages_per_row`）；截至撰寫此文，尚未實作。
- **格狀檢視的 PDF 編輯器**會列出一整面可捲動的縮圖牆 —— 用來重排頁面很方便，但那不是「每列固定幾頁、可調整」的閱讀版面。

PagesPerRow 補上的正是這個缺口：每列幾頁可自訂，為閱讀與綜覽而最佳化，而非為縮圖導覽。

## 功能

- **每列頁數可調** —— 1 到 8 的滑桿（預設 4）；格狀版面即時重排貼合。
- **獨立縮放** —— 10%–200% 的縮放滑桿，或 **fit-to-grid** 模式，依目前的欄數自動算出合適的縮放。隨時按 **Fit** 鈕切回自動。
- **單頁放大檢視** —— 點任一頁即可全螢幕放大看細節，且**文字可選取**（直接從 PDF 複製）。方向鍵（←/→）切換頁面；點頁面外的暗色區域或按 **Esc** 關閉。
- **全文搜尋** —— 在整份文件中尋找任意文字。命中的頁面會標上次數徽章，每個命中都會在頁面上高亮（格狀檢視與放大檢視都有），並可從工具列在命中之間跳轉。
- **Retina 清晰** —— 頁面以裝置像素比（device pixel ratio）渲染，放大時文字依然銳利。
- **本機、私密** —— 純前端、不上傳、無後端。PDF.js 從 CDN 載入；你的文件完全在瀏覽器內讀取。

> 狀態：早期 / 個人專案。為了搔自己讀論文的癢而做。歡迎回饋與 issue。

## 使用方式

**方式 A —— 直接開檔：**

1. 下載 [`index.html`](index.html)。
2. 用任何現代瀏覽器打開（Chrome、Firefox、Safari、Edge）。
3. 把 PDF 拖到視窗上，或點一下選檔。

**方式 B —— 自行託管：** 把 `index.html` 丟到任何靜態主機（GitHub Pages、Netlify、S3 bucket 等），把網址加進書籤即可。

兩種方式第一次都需要連網，因為 PDF.js 從 CDN 載入。

### 操作

| 動作 | 方式 |
|---|---|
| 載入 PDF | 把檔案拖到視窗上，或點放置區開啟檔案選擇 |
| 每列頁數 | 工具列的**每列頁數滑桿**（1–8） |
| 縮放 | **縮放滑桿**（10%–200%），或 **Fit** 鈕自動貼合 |
| 放大單頁 | 點該頁 |
| 切換頁面（放大檢視中） | **←** / **→** |
| 選取／複製文字（放大檢視中） | 照常拖曳劃過文字 |
| 關閉放大檢視 | **Esc**，或點頁面外的暗色區域 |
| 搜尋文字 | 在搜尋框輸入，或按 **Cmd/Ctrl+F** 聚焦 |
| 在命中之間跳轉 | **Enter** / **Shift+Enter**，或 **‹ ›** 按鈕 |

工具列會顯示檔名、目前的每列頁數與縮放值，以及總頁數。

## 未來方向

可能的發展方向（不保證實作）：

- 跳至指定頁碼的輸入框，以及鍵盤逐列捲動
- 各文件版面記憶（記住每個檔案的每列頁數與縮放）
- 超大 PDF 的延遲渲染（只渲染視窗內的頁面）
- Book mode（首頁單張，之後成對跨頁）
- 更聰明的搜尋：跨行／跨詞斷字匹配、正則、全字匹配
- 標註與螢光筆編輯 —— 目前刻意不在範圍內；這是綜覽／閱讀工具，不是編輯器。

## 技術

單一檔案的純 HTML/CSS/JS，無框架。渲染交給 [PDF.js](https://mozilla.github.io/pdf.js/)（從 cdnjs 以 ES module 載入）。頁面非同步渲染到 `<canvas>`、不阻塞 UI；當你變更縮放或欄數時，進行中的渲染會被取消。
